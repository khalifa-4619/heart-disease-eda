# Heart Disease Exploratory Data Analysis

## Project Overview
This project performs exploratory data analysis on a heart disease dataset to uncover
patterns and relationships between medical attributes and heart disease.

## Dataset
Source: UCI Heart Disease Dataset

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Chest pain type strongly correlates with heart disease
- Patients with lower max heart rate show higher risk
